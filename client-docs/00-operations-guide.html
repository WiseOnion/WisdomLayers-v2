Print Settings — Save Any Doc as PDF

Chrome (Recommended)

1

Open the HTML document in Chrome. Press Ctrl + P (Windows) or Cmd + P (Mac) to open Print dialog.

2

Destination: Click "Change" and select Save as PDF.

3

Pages: All

4

Paper size: Letter

5

Margins: Default (the documents already have their own 0.75in page margins built in)

6

Headers and footers: OFF. The documents have their own footers — browser headers will create duplicates.

7

Background graphics: ON (required). Without this, all teal headers, colored boxes, gradient buttons, and table headers will print as white/empty. Click "More settings" to find this option.

8

Click Save and choose your destination folder.

Background graphics must be ON for every document in this set. The teal header, colored info boxes, and table headers all require it. If your PDF looks washed out or white, this setting was missed.

Edge

Same as Chrome: Ctrl + P → Printer: Save as PDF → More settings → turn on Background graphics → turn off Headers and footers.

Firefox

Ctrl + P → Select "Microsoft Print to PDF" or "Save to PDF" → Options → check Print Background Colors and Images → uncheck Headers and Footers.

Which Docs Need "Background Graphics" ON

All 7 documents require background graphics ON. Every document has a teal gradient header, colored boxes, and/or table headers that will not appear without it.
