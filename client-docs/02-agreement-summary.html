Costs Not Included in Your Package Price

These are standard costs every website owner pays directly to outside companies — not to WisdomLayers:

  • Domain name (your web address, e.g. yourbusiness.com) — typically $10–$20/year
  • Website hosting (where your site lives online) — typically $5–$20/month
  • Any third-party software your site uses (booking systems, payment processors, etc.)

We can help you choose and set these up. The $35 domain setup fee covers our time to handle the technical side — the domain registration cost itself is separate.
