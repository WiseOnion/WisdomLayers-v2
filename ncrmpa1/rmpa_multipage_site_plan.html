Phase 3
Phase 1 — Launch Foundation

Core Website — All Essential Pages

  • Redesigned Home page (already done)
  • About / Mission, History stub, Administration
  • Membership / Join page with full form
  • Events calendar (static) + Monthly Meetings page
  • Gallery index + existing photos
  • Resources — Veteran Services + MP Links
  • Contact page with embedded map
  • Global nav / mega menu / mobile nav
  • SEO meta tags on all pages
  • Accessibility audit (WCAG 2.1 AA)
Estimated: 4–6 weeks (solo dev) or 2–3 weeks (team)
Phase 2 — Content & Engagement

Dynamic Content + Community Tools

  • CMS integration — officers can post news without coding
  • News / Announcements section (full blog)
  • Individual event pages with RSVP form
  • Gallery albums with lightbox viewer
  • Email newsletter signup (Mailchimp)
  • Document library with download tracking
  • Officer direct contact directory
  • Meeting minutes archive
Estimated: 4–8 weeks after Phase 1 launch
Phase 3 — Member Portal

Authenticated Member Features

  • Member login / account system
  • Searchable member directory (members only)
  • Online dues payment (Stripe)
  • Private document access
  • Scholarship application portal
  • Internal announcements board
Estimated: 6–10 weeks. Requires backend / auth infrastructure.
